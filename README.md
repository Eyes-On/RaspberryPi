## 적용한 방법

- 터미널에서 다음 파일 실행

```bash
sudo nano /etc/xdg/lxsession/LXDE-pi/autostart
```

- screensaver 윗줄에 입력

```bash
lxterminal -e python /home/pi/eyes-on/test.py
```
